Drug Effectiveness Prediction system built using machine learning on the Kaggle Drug Classification dataset. It takes basic clinical details of a patient—age, sex, blood pressure, cholesterol level, and sodium‑to‑potassium ratio—and predicts which of five drugs (Drug A, B, C, X, or Y) is likely to be most effective. The repository covers the full pipeline: data loading and preprocessing, training and comparing multiple classification models (Random Forest, Decision Tree, SVM, KNN, Logistic Regression, Naive Bayes), evaluating them with standard metrics, and selecting the best model. On top of that, it includes a simple Flask web interface where users can enter patient details in a form and get a real‑time drug recommendation along with a confidence score. The aim is to demonstrate how data mining and supervised learning can be combined to build a practical, end‑to‑end healthcare decision‑support prototype.
